## Vision

### 10x Check
The 10x version isn't just a benchmark harness — it's a **self-calibrating quality operating system** for AI-generated code. It learns which agent topology (solo, trio, trio+red-team) works per task type, produces replayable quality certificates, and becomes the standard way to prove AI code quality. Cross-model evaluator comparison gives confidence no single model's bias corrupts the results.

### Delight Opportunities Surfaced
1. Cross-model evaluator comparison (two models grade independently → disagreement = signal)
2. Benchmark difficulty auto-calibration (detect too-easy/too-hard benchmarks)
3. Replay mode (re-run evaluator on saved artifacts without re-generating)
4. HTML telemetry dashboard (shareable visual reports)
5. Pluggable criteria YAML (externalized taste profiles)
